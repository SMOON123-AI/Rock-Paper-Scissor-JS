# Rock Paper Scissors Game 🎮

A simple Rock Paper Scissors game built using **HTML, CSS, and JavaScript**.
Play against the computer and track your score in real time.

## Live Demo

https://smoon123-ai.github.io/Rock-Paper-Scissor-JS/

## Features

* Play Rock, Paper, Scissors against the computer
* Score tracking (Wins, Losses, Ties)
* Score saved using **localStorage**
* **Auto Play mode**
* **Keyboard shortcuts**

  * R → Rock
  * P → Paper
  * S → Scissors
* Clean and interactive UI

## Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

## How It Works

The computer randomly selects a move using `Math.random()`.
The game logic compares the player's move with the computer's move and updates the result and score.
